# Limitations

## Cannot Automate

- Editing `.github/workflows/` (protected by design)
- Files outside allowlist paths
- GitHub account settings (OAuth, billing, PATs)
- Repository settings (collaborators, webhooks)
- Actions that require human verification

## Can Automate

- All files in: `docs/`, `specs/`, `devices/`, `logs/`, `scripts/`
- Documentation updates
- Status tracking and progress
- Device registry management
- Specifications evolution
