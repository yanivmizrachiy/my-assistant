# Roadmap — "העוזרת שלי"

## עכשיו (Foundation)
- מקור אמת בגיטהאב: REQUIREMENTS / ROADMAP / CHANGELOG / STATUS
- כלי עדכון מהיר מ-Termux (commit+push)

- Device Registry + bootstrap לכל מכשיר
- UI בעברית (RTL) עם כפתורים והסברים
- Backend API (GitHub + Google + OpenAI)

## אוטומציה מלאה (Automation)

- Gmail label -> Issue אוטומטי
- Commit -> עדכון סטטוס אוטומטי
